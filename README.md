# Mohamed Ramy — Portfolio

An editorial, product-focused portfolio built around the **Organic Systems** brand direction: precise interface grids, an organic route, warm olive tones and a small signal-lime accent.

## Stack

- Next.js 16 App Router + TypeScript
- Tailwind CSS 4 with a custom visual system
- GSAP + `@gsap/react` + ScrollTrigger
- React Bits `Threads` background, adapted to the brand palette
- Next Image and Next Font

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
```

The homepage content is driven by `lib/portfolio-data.ts`. Project images live in `public/images/projects` and the downloadable résumé is `public/Mohamed-Ramy-CV.pdf`.
